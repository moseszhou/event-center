
# yum-event-manager

## Install

```bash
npm install yum-event-manager
```

## Usage

```js
import eventManger from 'yum-event-manager';

 const readyListener = eventManger.addEventListener("ready",(params)=>{ doing sth.});

 readyListener.remove()
 ```
